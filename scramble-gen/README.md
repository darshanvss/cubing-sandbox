# Cubing.js Sandbox

A sandbox project for experimenting with [cubing.js](https://github.com/cubing/cubing.js) - a library for displaying and working with twisty puzzles.

## Features

- Generate 3x3 scrambles
- 2D cube visualization with real-time updates
- Editable scramble input

## Getting Started

```bash
pnpm install
pnpm dev
```

Visit [http://localhost:3000](http://localhost:3000)

## Tech Stack

- Next.js 15
- React 19
- shadcn/ui
- cubing.js
- TypeScript
